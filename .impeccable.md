## Design Context

### Users
The interface is for an interactive course/demo installation about unusual user experience design. Users stand in front of a display, capture or upload an abstract image, and watch the system translate visual ambiguity into semantic words and three image generations.

### Brand Personality
Refined, artistic, calm. It should feel like a small digital exhibition tool rather than a generic web app.

### Aesthetic Direction
Use a fixed 4:3 presentation canvas with no page scrolling. The visual language should be editorial and gallery-like: restrained dark ink, warm paper, sharp geometric accents, compact text, and clear process feedback.

### Design Principles
1. Keep the file structure simple and the interaction obvious.
2. Make every screen fit inside a 4:3 camera/display ratio.
3. Show progress, but do not let users click progress steps directly.
4. Prioritize artistic interpretation: vision model extracts semantic imagination, then text-to-image produces high abstraction, semi-abstraction, and realism.
5. Avoid oversized text, noisy decoration, and fragile framework complexity.
